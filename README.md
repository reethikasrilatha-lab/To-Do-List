# ✅ To-Do List

A simple, responsive to-do list web app built using HTML, CSS, and JavaScript.

## 🔗 Live Demo
https://reethikasrilatha-lab.github.io/To-Do-list/
<!-- update this link once GitHub Pages is enabled -->

## 📌 Features
- Add new tasks to the list
- Mark tasks as complete
- Delete tasks from the list
- Clean and responsive UI

## 🛠️ Tech Stack
- **HTML** – structure of the app
- **CSS** – styling and layout
- **JavaScript** – task logic and DOM manipulation

## 🚀 How It Works
1. Type a task into the input box.
2. Click **Add** (or press Enter) to add it to the list.
3. Click a task to mark it as complete.
4. Click the delete icon/button to remove a task.
